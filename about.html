F4A2 Pemrograman Web

F4A2 Pemrograman Web merupakah sebuah mata kuliah yang ada di universitas ubhara jaya yang di pelajari mahasiswa semester 4 atau semester atas yang diajari oleh Bapak Achmad Noe'man, S.Kom., M.Kom. dan mempunyai kapasitas 35 mahasiswa, di universitas ubhara jaya juga mempunyai beberapa kelas dari kelas "A1 Sampai A7" maka dari itu setiap kelas hanya mempunyai 35 mahasiswa.

di kelas ini diajarkan tetang bagaimana membuat sebuah web dengan framework visual studio code, yang diajarkan pertama ialah tentang dasar-dasar html dan bagaimana mendesain menggunakan css dan juga dikelas ini sangat enak yang dimana penjelasannya sangat jelas dan menarik untuk supaya tidak merasa bosan

Materi yang diajarkan setiap pertemuan

  1. Menjelaskan Kontrak Perkuliahan dan menjelaskan RPS Pemrograman Web
  2. Menjelaskan HTML dan praktekan sintak HTML
  3. Membuat tabel di HTML
  4. Membuat Desain Web menggunakan HTML
  5. Membuat Desain Web menggunkan HTML
  6. Presentasi hasil desain web
